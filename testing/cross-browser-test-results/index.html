    Testing Instructions

    How to Test in Each Browser

    1. Install Browser (if not already installed)
    2. Firefox: https://www.mozilla.org/firefox/
    3. Edge: Built-in on Windows 10/11

    4. Safari: Mac OS only

    5. Open Prototype

    6. Navigate to: D:\Projects\see_pdfeditor\prototype\index.html
    7. Or use local server: python -m http.server 5173

    8. Or: npm run dev

    9. Run Test Cases

    10. Follow checklist above
    11. Mark ✅ for pass, ❌ for fail, ⚠️ for partial

    12. Document issues in "Browser-Specific Issues" section

    13. Report Bugs

    14. Include browser version
    15. Include OS version
    16. Include steps to reproduce
    17. Include screenshots if applicable
